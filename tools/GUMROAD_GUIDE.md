## 注册 + 获取API Key

**手机操作：**

1️⃣ 打开 https://gumroad.com （支持中文）

2️⃣ 点 Sign Up → 用你的QQ邮箱注册

3️⃣ 验证邮箱 → 完成注册

4️⃣ 进 Settings → Developer → Generate Access Token（勾选所有权限）

→ 得到一个类似 `xxxxxxxxxxxxxxxx` 的token

5️⃣ **把token发给我**，剩下的我全搞定：

   - 自动上架 Pro (¥49) 和 Enterprise (¥499) 两个产品

   - 设置自动交付凭证
